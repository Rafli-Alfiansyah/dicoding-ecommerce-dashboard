# E-Commerce Performance Analysis & Dashboard

This repository contains an end-to-end data analysis project utilizing the Olist Brazilian E-Commerce Public Dataset. The project encompasses data wrangling, exploratory data analysis (EDA), advanced customer segmentation using the RFM (Recency, Frequency, Monetary) framework, and an interactive, analytical companion dashboard built with Streamlit.

## Business Questions
1. **Product Performance:** Which top 10 product categories generated the highest total revenue during the full calendar year of 2018?
2. **Customer Segmentation:** What does the customer base look like when segmented with RFM (Recency, Frequency, Monetary) analysis using the last available 12 months of transactional data?

## Setup Environment - Anaconda
```
conda create --name main-ds python=3.9
conda activate main-ds
pip install -r requirements.txt
```

## Setup Environment - Shell/Terminal
```
mkdir olist_ecommerce_analysis
cd olist_ecommerce_analysis
pipenv install
pipenv shell
pip install -r requirements.txt
```

## Run steamlit app
```
streamlit run dashboard/dashboard.py
```
